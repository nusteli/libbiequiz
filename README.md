<h1>Image-Text Questions</h1>

Open the repo in browser (where you are right now!)

1. Click images/ folder
2. Click Add file → Upload files
3. Drag image(s) in
4. Click Commit changes
5. Getting the usable link (the key step)
> Click the uploaded image file in GitHub
> Follow this template: https://raw.githubusercontent.com/nusteli/libbiequiz/main/images/FILENAME.jpg
> >Replace FILENAME.jpg with your uploaded image filename and extension
6. Paste the full URL into QImage1 / QImage2 / QImage3 in Google Sheets

_TIP: If the link contains github.com/.../blob/... it’s wrong. It must be **raw**.githubusercontent.com/...._

Do **NOT** edit files in /config.

<h1>Text-Image Questions</h1>
1. Click on answer-images
2. If folder doesn't exist for your event, click on the three dots at the top (below Settings) > Upload files 
3. Drag ONE image file into the repository
4. Commit changes
5. Find your uploaded image in the answer-images folder
6. Click on the pencil icon to edit
7. In the file name box, put in a folder name (for your event) with a backslash, e.g. LibbiesBirthday/FILENAME.png
8. Commit changes
9. Now go to the event folder and upload the rest of the image files
10. To get the RAW link to the image, follow the template:
https://raw.githubusercontent.com/nusteli/libbiequiz/main/answer-images/FOLDERNAME/FILENAME.EXT
